# ToDoManager

Учебный Django-проект для управления задачами.

## Возможности

- Регистрация пользователей
- Авторизация и выход
- Создание задач
- Просмотр списка задач
- Редактирование задач
- Удаление задач
- Привязка задач к пользователю
- Поиск задач
- Фильтрация по статусу
- Сортировка задач

## Технологии

- Python
- Django
- PostgreSQL
- HTML
- Bootstrap
- Git
